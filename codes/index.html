Python

Python

Python هي لغة برمجية متعددة الاستخدامات وسهلة التعلم. تُستخدم في تطوير الويب، تحليل البيانات، والذكاء الاصطناعي.

مثال عملي:

# طباعة نص
print("مرحباً بالعالم!")

# جمع رقمين
def جمع(أ, ب):
    return أ + ب

print(جمع(3, 5))

شرح الكود:

الكود الأول يطبع نصاً على الشاشة، والكود الثاني يعرض دالة لجمع رقمين وإرجاع الناتج.

JavaScript

JavaScript

JavaScript هي لغة رئيسية لتطوير واجهات الويب التفاعلية، وتُستخدم في المتصفحات.

مثال عملي:

// طباعة نص
alert("مرحباً بالعالم!");

// جمع رقمين
function جمع(أ, ب) {
  return أ + ب;
}

console.log(جمع(3, 5));

شرح الكود:

الكود الأول يعرض رسالة تنبيهية في المتصفح، والكود الثاني يعرض دالة لجمع رقمين.

Java

Java

Java هي لغة قوية تُستخدم لتطوير التطبيقات الكبيرة والأنظمة المعقدة، بما في ذلك تطبيقات Android.

مثال عملي:

// طباعة نص
public class Main {
    public static void main(String[] args) {
        System.out.println("مرحباً بالعالم!");
    }
}

// جمع رقمين
public int جمع(int أ, int ب) {
    return أ + ب;
}

شرح الكود:

الكود الأول يطبع نصاً على الشاشة، والكود الثاني يعرض دالة لجمع رقمين.

C++

C++

C++ هي لغة عالية الأداء تُستخدم لتطوير الألعاب والأنظمة.

مثال عملي:

#include <iostream>
using namespace std;

int main() {
    cout << "مرحباً بالعالم!" << endl;
    return 0;
}

// جمع رقمين
int جمع(int أ, int ب) {
    return أ + ب;
}

شرح الكود:

الكود يطبع نصاً باستخدام مكتبة iostream ويعرض دالة لجمع رقمين.
