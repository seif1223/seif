ما هي لغة بايثون؟

بايثون هي لغة برمجة متعددة الاستخدامات وسهلة التعلم تُستخدم في تطوير التطبيقات، تحليل البيانات، الذكاء الاصطناعي، وغير ذلك.

أساسيات بايثون

  • المتغيرات: لتخزين البيانات. مثال: x = 10
  • الحلقات: لتكرار العمليات. مثال: for i in range(5): print(i)
  • الشروط: لاتخاذ القرارات. مثال: if x > 5: print("x أكبر من 5")

مثال عملي

# برنامج بسيط لطباعة الأرقام
for i in range(1, 6):
    print(f"الرقم هو: {i}")
